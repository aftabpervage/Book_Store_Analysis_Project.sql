# Book Store Sales Analysis using SQL

## End-to-End SQL Data Analytics Project

A complete SQL-based Data Analytics project focused on analyzing bookstore sales data to generate meaningful business insights related to customer purchases, book categories, revenue trends, and sales performance.

This project demonstrates the use of SQL for data cleaning, querying, aggregation, and business analysis using real-world retail bookstore data.

---

# Project Overview

The main objective of this project is to analyze bookstore sales data and identify important trends, customer behavior, and revenue-generating opportunities using SQL queries.

The analysis helps in understanding:

- Best-selling books
- Revenue performance
- Customer purchasing trends
- Category-wise sales analysis
- Book pricing analysis
- Order and sales insights
- Inventory-related observations
